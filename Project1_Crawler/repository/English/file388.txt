         </h2>
         <div class="mainpage-content-text">
          <div class="floatleft">
           <a class="image" href="/wiki/%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB:Cholesterol.svg">
            <img alt="Cholesterol.svg" data-file-height="296" data-file-width="440" decoding="async" height="101" src="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/Cholesterol.svg/150px-Cholesterol.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/9/9a/Cholesterol.svg/225px-Cholesterol.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/9/9a/Cholesterol.svg/300px-Cholesterol.svg.png 2x" width="150"/>
           </a>
          </div>
          <p>
           <b>
            <a href="/wiki/%E3%82%B3%E3%83%AC%E3%82%B9%E3%83%86%E3%83%AD%E3%83%BC%E3%83%AB" title="コレステロール">
             コレステロール
            </a>
           </b>
           は
           <a href="/wiki/%E3%82%B9%E3%83%86%E3%83%AD%E3%82%A4%E3%83%89" title="ステロイド">
            ステロイド
           </a>
           に分類され、その中でも
           <a href="/wiki/%E3%82%B9%E3%83%86%E3%83%AD%E3%83%BC%E3%83%AB" title="ステロール">
            ステロール
           </a>
           と呼ばれるサブグループに属する
           <a href="/wiki/%E6%9C%89%E6%A9%9F%E5%8C%96%E5%90%88%E7%89%A9" title="有機化合物">
            有機化合物
           </a>
           の一種である。
           <a href="/wiki/1784%E5%B9%B4" title="1784年">
            1784年
           </a>
           に研究者が
           <a href="/wiki/%E8%83%86%E7%9F%B3" title="胆石">
            胆石
           </a>
           からコレステロールが初めて
           <a href="/wiki/%E5%8D%98%E9%9B%A2" title="単離">
            単離
           </a>
           された。室温で単離された場合は白色ないしは微黄色の固体である。生体内では
           <a href="/wiki/%E3%82%B9%E3%82%AF%E3%82%A2%E3%83%AC%E3%83%B3" title="スクアレン">
            スクアレン
           </a>
           から
           <a href="/wiki/%E3%83%A9%E3%83%8E%E3%82%B9%E3%83%86%E3%83%AD%E3%83%BC%E3%83%AB" title="ラノステロール">
            ラノステロール
           </a>
           を経て
           <a href="/wiki/%E7%94%9F%E5%90%88%E6%88%90" title="生合成">
            生合成
           </a>
           される。
          </p>
          <p>
           コレステロール分子自体は、動物
           <a href="/wiki/%E7%B4%B0%E8%83%9E" title="細胞">
            細胞
           </a>
           にとっては
           <a href="/wiki/%E7%94%9F%E4%BD%93%E8%86%9C" title="生体膜">
            生体膜
           </a>
           の構成物質であったり、さまざまな生命現象に関わる重要な化合物である。よって生体において広く分布しており、主要な生体分子といえる。精製物は、化粧品・医薬品・
           <a href="/wiki/%E6%B6%B2%E6%99%B6" title="液晶">
            液晶
           </a>
           の原材料など工業原料としても利用される。……
          </p>

              つまみ読み

            </li>

            <a href="/wiki/%E5%A5%88%E8%89%AF%E7%9C%8C" title="奈良県">

           </li>
           <li>
            <b>
             <a href="/wiki/%E4%BC%8A%E8%A8%88%E5%B3%B6" title="伊計島">

            </b>

            <a href="/wiki/%E6%B2%96%E7%B8%84%E7%9C%8C" title="沖縄県">
             沖縄県
            </a>
            <a href="/wiki/%E3%81%86%E3%82%8B%E3%81%BE%E5%B8%82" title="うるま市">
             うるま市
            </a>
            に属する
            <a href="/wiki/%E5%B3%B6" title="島">
             島
            </a>

            <a href="/wiki/%E6%B2%96%E7%B8%84%E6%9C%AC%E5%B3%B6" title="沖縄本島">
             沖縄本島
            </a>
            中部の東部海岸に突出する
            <a class="mw-redirect" href="/wiki/%E5%8B%9D%E9%80%A3%E5%8D%8A%E5%B3%B6" title="勝連半島">

             琉球石灰岩
            </a>
            に覆われている。2012年4月時点での島内人口は318人である……
           </li>
           <li>
            <b>
             <a href="/wiki/%E3%83%AA%E3%83%BC%E3%82%BC%E3%83%BB%E3%83%9E%E3%82%A4%E3%83%88%E3%83%8A%E3%83%BC" title="リーゼ・マイトナー">
              リーゼ・マイトナー
             </a>
