               <p>
                <b>
                 <a href="/wiki/%D3%98%D1%85%D0%BC%D3%99%D1%82%D0%B7%D3%99%D0%BA%D0%B8_%D0%92%D3%99%D0%BB%D0%B8%D0%B4%D0%B8_%D0%A2%D1%83%D2%93%D0%B0%D0%BD" title="Әхмәтзәки Вәлиди Туған">
                  Әхмәтзәки Вәлиди Туған
                 </a>
                </b>
                (
                <a href="/wiki/10_%D0%B4%D0%B5%D0%BA%D0%B0%D0%B1%D1%80%D1%8C" title="10 декабрь">
                 10 декабрь
                </a>
                <a href="/wiki/1890_%D0%B9%D1%8B%D0%BB" title="1890 йыл">
                 1890 йыл
                </a>
                —
                <a href="/wiki/26_%D0%B8%D1%8E%D0%BB%D1%8C" title="26 июль">
                 26 июль
                </a>
                <a href="/wiki/1970_%D0%B9%D1%8B%D0%BB" title="1970 йыл">
                 1970 йыл
                </a>
                ) — күренекле сәйәсмән һәм дәүләт эшмәкәре,
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D0%BC%D0%B8%D0%BB%D0%BB%D0%B8_%D1%85%C9%99%D1%80%C9%99%D0%BA%C9%99%D1%82%D0%B5" title="Башҡорт милли хəрəкəте">
                 Башҡорт милли-азатлыҡ хəрəкəте
                </a>
                етəксеһе,
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82%D0%BE%D1%81%D1%82%D0%B0%D0%BD_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F%D2%BB%D1%8B" title="Башҡортостан автономияһы">
                 автономиялы Башҡортостанға
                </a>
                нигеҙ һалыусы. Шәрҡиәтсе һәм төркиәт белгесе, Фәлсәфә докторы (1935), профессор, Манчестер университетының мөхбир докторы (1967).
               </p>
               <p>
                Әхмәтзәки Вәлиди 1890 йылдың 10 декабрендә
                <a href="/wiki/%D3%A8%D1%84%D3%A9_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B0%D2%BB%D1%8B" title="Өфө губернаһы">
                 Өфө губернаһы
                </a>
                <a href="/wiki/%D0%A1%D1%82%D3%99%D1%80%D0%BB%D0%B5%D1%82%D0%B0%D0%BC%D0%B0%D2%A1_%D3%A9%D0%B9%D3%99%D2%99%D0%B5" title="Стәрлетамаҡ өйәҙе">
                 Стәрлетамаҡ өйәҙе
                </a>
                Илсек-Тимер улусының, хәҙерге
                <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82%D0%BE%D1%81%D1%82%D0%B0%D0%BD_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B0%D2%BB%D1%8B" title="Башҡортостан Республикаһы">
                 Башҡортостан Республикаһы
                </a>
                <a href="/wiki/%D0%98%D1%88%D0%B5%D0%BC%D0%B1%D0%B0%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD%D1%8B" title="Ишембай районы">
                 Ишембай районының
                </a>
                <a href="/wiki/%D0%9A%D3%A9%D2%99%D3%99%D0%BD_(%D0%98%D1%88%D0%B5%D0%BC%D0%B1%D0%B0%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD%D1%8B)" title="Көҙән (Ишембай районы)">
                 Көҙән
                </a>
                ауылында тыуған.
               </p>
               <p>
                1917 йылда социал-революционерҙар (эсерҙар) партияһына инә. Шул уҡ йылдың март — апрелендә Рəсəй мосолмандарының ваҡытлы үҙəк бюроһы ағзаһы була, 1917 йылдың апрелендә
                <a href="/wiki/%D0%A2%D0%B0%D1%88%D0%BA%D0%B5%D0%BD%D1%82" title="Ташкент">
                 Ташкент
                </a>
                ҡалаһында Мосолман советын ойоштороуҙа ҡатнаша һәм унда шул уҡ йылдың майында
                <a href="/wiki/%D0%9C%D3%99%D1%81%D0%BA%D3%99%D2%AF" title="Мәскәү">
                 Мәскәүҙә
                </a>
                уҙғарылған I Бөтə Рəсəй мосолман съезына Төркөстандан делегат итеп һайлана. Был съезда Әхмәтзәки Вәлиди Рəсəйҙең федератив республикаһының ҡоролошо һəм мосолман халыҡтарының милли-территориаль автономияһы идеяһын яҡлап сыға һәм Бөтə Рəсəй мосолман шураһының башҡарма комитеты һәм
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D3%A9%D0%BB%D0%BA%D3%99_%D0%B1%D1%8E%D1%80%D0%BE%D2%BB%D1%8B" title="Башҡорт өлкә бюроһы">
                 Башҡорт өлкә бюроһы
                </a>
                ағзаһы булып китә.
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_(%D0%B3%D3%99%D0%B7%D0%B8%D1%82)" title="Башҡорт (гәзит)">
                 «Башҡорт»
                </a>
                гәзитенең тәүге һанында Вәлидиҙең Башҡортостан үҙаллығы тураһында мәҡәләһе баҫылып сыға.
               </p>
               <p>
                1917 йылдың 20—27 июлендә
                <a href="/wiki/%D0%AB%D1%80%D1%8B%D0%BC%D0%B1%D1%83%D1%80" title="Ырымбур">
                 Ырымбурҙа
                </a>
                үткән
                <a href="/wiki/%D0%91%D3%A9%D1%82%D3%99_%D0%B1%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D2%A1%D0%BE%D1%80%D0%BE%D0%BB%D1%82%D0%B0%D0%B9%D2%99%D0%B0%D1%80%D1%8B" title="Бөтә башҡорт ҡоролтайҙары">
                 I Башҡорт ҡоролтайында
                </a>
                ҡатнаша һәм унда президиумы ағзаһы һәм
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D0%BC%D3%99%D1%80%D0%BA%D3%99%D0%B7_%D1%88%D1%83%D1%80%D0%B0%D2%BB%D1%8B" title="Башҡорт мәркәз шураһы">
                 Башҡорт мәркәз шураһы
                </a>
                рəйесе урынбаҫары итеп һайлана. Дəүлəт ҡоролошо (милли-территориаль автономия нигеҙендə) һəм «Ер мәсьәләләре» буйынса докладтар менән сығыш яһай. Икенсе докладында, ул XVII—XX быуаттар араһында юғалтылған башҡорт ерҙəрен ҡайтарып биреү зарурлығын билдəлəп үтә. 1917 йылдың 22—29 авгусында
                <a href="/wiki/%D3%A8%D1%84%D3%A9" title="Өфө">
                 Өфөлә
                </a>
                уҙған
                <a href="/wiki/%D0%91%D3%A9%D1%82%D3%99_%D0%B1%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D2%A1%D0%BE%D1%80%D0%BE%D0%BB%D1%82%D0%B0%D0%B9%D2%99%D0%B0%D1%80%D1%8B" title="Бөтә башҡорт ҡоролтайҙары">
                 II Башҡорт ҡоролтайында
                </a>
                ҡатнаша. Бында ул «
                <a href="/wiki/%D0%9E%D0%BB%D0%BE_%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82%D0%BE%D1%81%D1%82%D0%B0%D0%BD" title="Оло Башҡортостан">
                 Оло Башҡортостандың»
                </a>
                , «
                <a href="/wiki/%D0%91%D3%99%D0%BB%D3%99%D0%BA%D3%99%D0%B9_%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82%D0%BE%D1%81%D1%82%D0%B0%D0%BD" title="Бәләкәй Башҡортостан">
                 Бәләкәй Башҡортостандың»
                </a>
                һəм «Көнсығыш Рəсəйҙең Мосолман автономлы дəүлəттəре федерацияһының» карталарын төҙөй.
                <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D1%85%D3%A9%D0%BA%D2%AF%D0%BC%D3%99%D1%82%D0%B5" title="Башҡорт хөкүмәте">
                 Башҡорт хөкүмәте
                </a>
                төҙөлгәс, Вәлидигә Хәрби һәм эске эштәр бүлектәре тапшырыла.
               </p>
               <p>
                1917 йылдың ноябрендә Өфө губернаһының федералист башҡорттар исемлеге буйынса
                <a class="mw-redirect" href="/wiki/%D0%91%D3%A9%D1%82%C9%99_%D0%A0%C9%99%D1%81%C9%99%D0%B9_%D0%9E%D0%B9%D0%BE%D1%88%D1%82%D0%BE%D1%80%D0%BE%D1%83_%D0%B9%D1%8B%D0%B9%D1%8B%D0%BB%D1%8B%D1%88%D1%8B" title="Бөтə Рəсəй Ойоштороу йыйылышы">
                 Бөтə Рəсəй Ойоштороу йыйылышына
                </a>
                делегат итеп һайлана. Башҡорт мəркəз шураһының беренсе фармандарын төҙөүҙə һәм уларға ҡул ҡуйыуҙа ҡатнаша. Был фармандарҙа башҡорттарҙың сəйəси көрəштə позицияһы һəм федератив Рəсəй составында
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82%D0%BE%D1%81%D1%82%D0%B0%D0%BD_%D0%B0%D0%B2%D1%82%D0%BE%D0%BD%D0%BE%D0%BC%D0%B8%D1%8F%D2%BB%D1%8B" title="Башҡортостан автономияһы">
                 Башҡортостан автономияһы
                </a>
                иғлан ителә. 1917 йылдың 8—20 декабрендә Ырымбурҙа үткән
                <a href="/wiki/%D0%91%D3%A9%D1%82%D3%99_%D0%B1%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D2%A1%D0%BE%D1%80%D0%BE%D0%BB%D1%82%D0%B0%D0%B9%D2%99%D0%B0%D1%80%D1%8B" title="Бөтә башҡорт ҡоролтайҙары">
                 III Башҡорт ҡоролтайында
                </a>
                ҡатнаша һәм унда президиумы рәйесе һәм
                <a href="/wiki/%D0%9A%D0%B5%D1%81%D0%B5_%D2%A0%D0%BE%D1%80%D0%BE%D0%BB%D1%82%D0%B0%D0%B9" title="Кесе Ҡоролтай">
                 Кесе Ҡоролтай
                </a>
                ағзаһы итеп һайлана,
                <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D1%85%D3%A9%D0%BA%D2%AF%D0%BC%D3%99%D1%82%D0%B5" title="Башҡорт хөкүмәте">
                 Башҡорт хөкүмәте
                </a>
                рəйесе урынбаҫары һəм Хəрби бүлек мөдире итеп раҫлана.
               </p>
               <p>
                1918 йылдың июненән алып 1919 йылдың февраленә тиклем
                <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D1%85%C9%99%D1%80%D0%B1%D0%B8_%D1%88%D1%83%D1%80%D0%B0%D2%BB%D1%8B" title="Башҡорт хəрби шураһы">
                 Башҡорт хəрби шураһы
                </a>
                рəйесе,
                <a class="mw-redirect" href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D0%B0%D1%80%D0%BC%D0%B8%D1%8F%D2%BB%D1%8B" title="Башҡорт армияһы">
                 Башҡорт ғәскәре
                </a>
                командующийы була. Башҡорт хөкүмəтенең

                <a href="/wiki/%D3%A8%D1%84%D3%A9_%D0%B4%D3%99%D2%AF%D0%BB%D3%99%D1%82_%D0%BA%D3%99%D2%A3%D3%99%D1%88%D0%BC%D3%99%D2%BB%D0%B5" title="Өфө дәүләт кәңәшмәһе">

               <p>
                <b>
                 <a href="/wiki/%D2%A0%D0%B0%D2%A3%D0%BB%D1%8B" title="Ҡаңлы">
                  Ҡаңлы
                 </a>
                </b>
                (
                <i>
                 ҡаңны
                </i>
                ) —
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D2%A1%D3%99%D0%B1%D0%B8%D0%BB%D3%99%D0%BB%D3%99%D1%80%D0%B5" title="Башҡорт ҡәбиләләре">
                 башҡорт ҡәбиләһе
                </a>
                .
               </p>
               <p>
                Телдәре
                <a href="/wiki/%D0%91%D0%B0%D1%88%D2%A1%D0%BE%D1%80%D1%82_%D1%82%D0%B5%D0%BB%D0%B5" title="Башҡорт теле">
                 башҡорт теленең
                </a>
                <a href="/wiki/%D0%A2%D3%A9%D0%BD%D1%8C%D1%8F%D2%A1-%D0%BA%D3%A9%D0%BD%D0%B1%D0%B0%D0%B9%D1%8B%D1%88_%D0%B4%D0%B8%D0%B0%D0%BB%D0%B5%D0%BA%D1%82" title="Төньяҡ-көнбайыш диалект">
                 төньяҡ-көнбайыш диалекты
                </a>
                <a href="/wiki/%D0%A2%D2%AF%D0%B1%D3%99%D0%BD%D0%B3%D0%B5_%D0%B0%D2%93%D0%B8%D2%99%D0%B5%D0%BB-%D1%8B%D2%A1_%D2%BB%D3%A9%D0%B9%D0%BB%D3%99%D1%88%D0%B5" title="Түбәнге ағиҙел-ыҡ һөйләше">
                 түбәнге ағиҙел-ыҡ
                </a>
                менән
                <a href="/wiki/%D0%A2%D0%B0%D0%BD%D1%8B%D0%BF_%D2%BB%D3%A9%D0%B9%D0%BB%D3%99%D1%88%D0%B5" title="Танып һөйләше">
                 танып
                </a>
                һәм
                <a href="/wiki/%D0%9A%D3%A9%D0%BD%D1%8C%D1%8F%D2%A1_%D0%B4%D0%B8%D0%B0%D0%BB%D0%B5%D0%BA%D1%82" title="Көньяҡ диалект">
                 көньяҡ диалекттың
                </a>
                <a href="/wiki/%D0%94%D0%B8%D0%BC_%D2%BB%D3%A9%D0%B9%D0%BB%D3%99%D1%88%D0%B5" title="Дим һөйләше">
                 дим
                </a>
                һөйләштәренә ҡарай.
               </p>
               <p>
                Ҡаңлы ҡәбиләһенең барлыҡҡа килеүен боронғо Туран халҡы менән бәйләйҙәр, улар актик һәм иртә урта быуаттарҙа
                <a href="/wiki/%D2%BA%D1%8B%D1%80%D0%B4%D0%B0%D1%80%D1%8A%D1%8F" title="Һырдаръя">
                 Һырдаръя
                </a>
                һәм
                <a href="/wiki/%D0%9A%D3%A9%D0%BD%D1%8C%D1%8F%D2%A1_%D0%A3%D1%80%D0%B0%D0%BB" title="Көньяҡ Урал">
                 Көньяҡ Урал
                </a>
                араһындағы далаларҙа йәшәгән. Боронғо иран ижади ҡомартҡыларында («
                <a href="/wiki/%D0%90%D0%B2%D0%B5%D1%81%D1%82%D0%B0" title="Авеста">
                 Авеста»
                </a>
                , «Хвадай-намаг» («Хоҙайнамә»)), зороастрий яҙмаларында («Денкард», «Бундахишн»), «
                <a href="/wiki/%D0%A8%D0%B0%D2%BB%D0%BD%D0%B0%D0%BC%D3%99" title="Шаһнамә">
                 Шаһнамә»
                </a>
                поэмаһында туран һәм иран (арий) халыҡтарының ҡан-ҡәрҙәш булыуы тураһында әйтелә. Тәүге турандарҙың мифик ата-бабаһы Арийҙың ҡустыһы булған, был хәҙерге генетик тикшеренеүҙәр менән дә раҫлана.
               </p>
               <p>
                Урта быуат ҡаңлылары «Сокровенное сказание»ла — канг-лин,
                <a class="mw-redirect" href="/wiki/%D0%A0%D3%99%D1%88%D0%B8%D1%82_%D3%99%D1%82-%D0%94%D0%B8%D0%BD" title="Рәшит әт-Дин">
                 Рәшит әт-Диндә
                </a>
                — канлы,
                <a href="/wiki/%D3%98%D0%B1%D2%AF_%D3%99%D0%BB-%D2%92%D0%B0%D0%B7%D0%B8" title="Әбү әл-Ғази">
                 Әбү әл-Ғази
                </a>
                ла канкли, канглы тип телгә алына. Был этноним шулай уҡ
                <a href="/wiki/%D2%A0%D0%B0%D2%99%D0%B0%D2%A1%D1%82%D0%B0%D1%80" title="Ҡаҙаҡтар">
                 ҡаҙаҡтар
                </a>
                ,
                <a class="mw-redirect" href="/wiki/%D2%A0%D0%B0%D1%80%D0%B0%D2%A1%D0%B0%D0%BB%D0%BF%D0%B0%D2%A1%D1%82%D0%B0%D1%80" title="Ҡараҡалпаҡтар">
                 ҡараҡалпаҡтар
                </a>
                ,
                <a href="/wiki/%D2%A0%D1%8B%D1%80%D2%93%D1%8B%D2%99%D2%99%D0%B0%D1%80" title="Ҡырғыҙҙар">
                 ҡырғыҙҙар
                </a>
                ,
                <a href="/wiki/%D2%A0%D1%8B%D1%80%D1%8B%D0%BC_%D1%82%D0%B0%D1%82%D0%B0%D1%80%D2%99%D0%B0%D1%80%D1%8B" title="Ҡырым татарҙары">
                 ҡырым татарҙары
                </a>
                ,
                <a href="/wiki/%D0%9D%D1%83%D2%93%D0%B0%D0%B9%D2%99%D0%B0%D1%80" title="Нуғайҙар">
                 нуғайҙар
                </a>
                һәм
                <a href="/wiki/%D2%AE%D0%B7%D0%B1%D3%99%D0%BA%D1%82%D3%99%D1%80" title="Үзбәктәр">
                 үзбәктәрҙең
                </a>
                составында теркәлгән. Ҡәбилә формалашыуына уларҙың
                <a href="/wiki/%D2%A0%D1%8B%D0%BF%D1%81%D0%B0%D2%A1%D1%82%D0%B0%D1%80" title="Ҡыпсаҡтар">
                 ҡыпсаҡтар
                </a>
                араһында йәшәүе йоғонто яһай. Тикшеренеүселәрҙең күбеһенең раҫлауынса, ҡаңлыларҙың берләшмәһе XI—XII быуаттарҙа
                <a href="/wiki/%D0%90%D1%80%D0%B0%D0%BB_%D0%B4%D0%B8%D2%A3%D0%B3%D0%B5%D2%99%D0%B5" title="Арал диңгеҙе">
                 Арал
                </a>
                буйы далаларында
                <a href="/wiki/%D0%91%D3%99%D1%88%D0%BD%D3%99%D0%BA%D1%82%D3%99%D1%80" title="Бәшнәктәр">
                 бәшнәк
                </a>
                -
                <a href="/wiki/%D0%A3%D2%93%D1%8B%D2%99%D2%99%D0%B0%D1%80" title="Уғыҙҙар">
                 уғыҙ
                </a>
                нигеҙендә барлыҡҡа килгән. Шул уҡ ваҡытта «ҡаңлы» этнонимы бәшнәктең боронғо үҙатамаһы булған «ҡаңғар» менән оҡшаш. Ҡаңлыларҙың этник формалашыуында шулай уҡ ҡыпсаҡтарҙың ҙур роль уйнауы һыҙыҡ өҫтөнә алына, улар 1220 йылда
                <a href="/wiki/%D0%9C%D0%BE%D0%BD%D0%B3%D0%BE%D0%BB%D0%B4%D0%B0%D1%80" title="Монголдар">

                </a>
                тарафынан тар-мар ителгәндән һуң башлыса
                <a class="mw-redirect" href="/wiki/%D2%A0%D1%8B%D0%BF%D1%81%D0%B0%D2%A1_%D0%B4%D0%B0%D0%BB%D0%B0%D2%BB%D1%8B" title="Ҡыпсаҡ далаһы">
                 Ҡыпсаҡ далаһы
                </a>
                (Дешт-и-Ҡыпсаҡ) халҡы составына ҡушыла. ­Икенсе гипотеза буйынса, ҡаңлылар
                <a href="/wiki/%D2%BA%D1%8B%D1%80%D0%B4%D0%B0%D1%80%D1%8A%D1%8F" title="Һырдаръя">
                 Һырдаръяла
                </a>
                Ҡаңгөй дәүләтен төҙөгән ҡаңгөйҙәрҙең этник вариҫтары булып тора.
               </p>
               <p>
                Ҡаңлылар
                <a href="/wiki/%D0%90%D1%81%D1%8B%D0%BB%D1%8B%D0%BA%D2%AF%D0%BB" title="Асылыкүл">
                 Асылыкүл
                </a>
                менән

                </a>
                буйында,

                </a>
                йылғаһы бассейны, шулай уҡ

                <a href="/wiki/%D3%A8%D2%AB%D3%99%D0%BD" title="Өҫән">

                </a>
                йылғаһының — урта,

                 Ағиҙел
                </a>
                йылғаһының түбәнге ағымында төпләнә, бында улар
                <a href="/wiki/%D0%94%D1%8B%D1%83%D0%B0%D0%BD" title="Дыуан">
                 дыуан
                </a>
                ,
                <a href="/wiki/%D0%99%D1%8B%D0%BB%D0%B0%D0%BD" title="Йылан">

                ,

                 йәлдәк
                </a>
                ,
                <a href="/wiki/%D2%A0%D0%B0%D1%80%D1%88%D1%8B%D0%BD" title="Ҡаршын">
                 ҡаршын
                </a>
                ,
                <a class="new" href="/w/index.php?title=%D2%A0%D0%BE%D0%B1%D0%B0%D1%83&amp;action=edit&amp;redlink=1" title="Ҡобау (был бит юҡ)">
                 ҡобау
                </a>
                ,
                <a href="/wiki/%D0%9C%D0%B5%D2%A3" title="Мең">
                 мең

                һәм
                <a href="/wiki/%D0%A3%D0%BD%D0%BB%D0%B0%D1%80" title="Унлар">

                </a>
                ҡәбиләләре менән күрше була.
                <a href="/wiki/%D0%9A%D1%83%D0%B7%D0%B5%D0%B5%D0%B2_%D0%A0%D3%99%D0%B8%D0%BB_%D2%92%D2%AF%D0%BC%D3%99%D1%80_%D1%83%D0%BB%D1%8B" title="Кузеев Рәил Ғүмәр улы">
