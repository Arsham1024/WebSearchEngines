        <h2>
         <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Get_involved","replies":[],"headingLevel":2,"placeholderHeading":false}' id="Get_involved">
          <span data-mw-comment-start="" id="h-Get_involved">
          </span>
          Get involved
          <span data-mw-comment-end="h-Get_involved">
          </span>
         </span>
        </h2>
        <p>
         Do you love software freedom and
         <a class="extiw" href="https://en.wikipedia.org/wiki/Open_source" title="w:Open source">
          open source communities
         </a>
         ? Do you like the
         <a href="/wiki/Special:MyLanguage/MediaWiki" title="Special:MyLanguage/MediaWiki">
          MediaWiki
         </a>
         software,
         <a class="external text" href="https://www.wikipedia.org/">
          Wikipedia
         </a>
         , or any other
         <a class="external text" href="https://www.wikimedia.org/">
          Wikimedia sites
         </a>
         ?
        </p>
        Then, contribute your skills here and learn from other contributors. This page will help you get started by providing an overview of areas where you can get involved.

       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-More_helpful_information","replies":["h-Communication-More_helpful_information","h-Editing_and_discussing_in_MediaWiki-More_helpful_information"],"headingLevel":2,"placeholderHeading":false}' id="More_helpful_information">
        <span data-mw-comment-start="" id="h-More_helpful_information">
        </span>
        More helpful information
        <span data-mw-comment-end="h-More_helpful_information">
        </span>
       </span>
      </h2>
      <h3>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Communication-More_helpful_information","replies":[],"headingLevel":3,"placeholderHeading":false}' id="Communication">
        <span data-mw-comment-start="" id="h-Communication-More_helpful_information">
        </span>
        Communication
        <span data-mw-comment-end="h-Communication-More_helpful_information">
        </span>
       </span>
      </h3>
      <p>
       There are several ways you can get into
       <a href="/wiki/Special:MyLanguage/Communication" title="Special:MyLanguage/Communication">
        contact
       </a>
       with the Wikimedia community.
      </p>
      <ul>
       <li>
        For
        <a href="/wiki/Special:MyLanguage/MediaWiki_on_IRC" title="Special:MyLanguage/MediaWiki on IRC">
         real-time communication
        </a>
        use
        <span class="plainlinks" style="font-family: monospace,Courier; white-space: pre-wrap !important; word-wrap: break-word; max-width: 1200px; overflow: auto;" title="#wikimedia-dev on Libera.Chat IRC">
         <a class="external text" href="irc://libera.chat:6667/wikimedia-dev" rel="nofollow">
          #wikimedia-dev
         </a>
        </span>
        <sup class="plainlinks">
         <a class="external text" href="https://web.libera.chat/?channel=#wikimedia-dev" rel="nofollow">
          <span style="color:green;">
           connect
          </span>
         </a>
        </sup>
        on
        <a class="extiw" href="https://en.wikipedia.org/wiki/Internet_Relay_Chat" title="w:Internet Relay Chat">
         IRC (Internet Relay Chat)
        </a>
        .
       </li>
       <li>
        There are also several
        <a href="/wiki/Special:MyLanguage/Mailing_lists" title="Special:MyLanguage/Mailing lists">
         email lists
        </a>
        for discussion of Wikimedia software. A
        <a class="extiw" href="https://meta.wikimedia.org/wiki/Special:MyLanguage/Mailing_lists/Overview#MediaWiki" title="m:Special:MyLanguage/Mailing lists/Overview">
         complete list
        </a>
        is available.
       </li>
       <li>
        You can also follow and share Wikimedia news across your
        <a href="/wiki/Special:MyLanguage/Social_media" title="Special:MyLanguage/Social media">
         social network
        </a>
        .
       </li>
       <li>
        You can also subscribe to
        <a class="extiw" href="https://meta.wikimedia.org/wiki/Tech/News" title="m:Tech/News">
         Tech News
        </a>
        to receive a weekly summary on your user talk page of recent software changes, without technical jargon.
       </li>
      </ul>
      <h3>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Editing_and_discussing_in_MediaWiki-More_helpful_information","replies":[],"headingLevel":3,"placeholderHeading":false}' id="Editing_and_discussing_in_MediaWiki">
        <span data-mw-comment-start="" id="h-Editing_and_discussing_in_MediaWiki-More_helpful_information">
        </span>
        Editing and discussing in MediaWiki
        <span data-mw-comment-end="h-Editing_and_discussing_in_MediaWiki-More_helpful_information">
        </span>
       </span>
      </h3>
      <p>
       If you have not used MediaWiki before:
      </p>
      <ul>
       <li>
        <a href="/wiki/Special:UserLogin" title="Special:UserLogin">
         Register your user account on mediawiki.org
        </a>
        .
       </li>
       <li>
        <a href="/wiki/Special:MyLanguage/Help:VisualEditor/User_guide" title="Special:MyLanguage/Help:VisualEditor/User guide">
         Learn how to edit
        </a>
        wiki pages with VisualEditor or
        <a href="/wiki/Special:MyLanguage/Help:Editing_pages" title="Special:MyLanguage/Help:Editing pages">
         editing the source
        </a>
        .
       </li>
       <li>
        Feel free to edit
        <a href="/wiki/Special:MyPage" title="Special:MyPage">
         your public user page
        </a>
        . Introduce yourself. You may use the
        <a href="/wiki/Template:User_info" title="Template:User info">
         User Info Template
        </a>
        . Learn more reading the
        <a class="extiw" href="https://en.wikipedia.org/wiki/Wikipedia:User_pages" title="w:Wikipedia:User pages">
         Wikipedia instructions
        </a>
        .
       </li>
       <li>
        Check
        <a href="/wiki/Special:MyLanguage/Help:Navigation" title="Special:MyLanguage/Help:Navigation">
         Help:Navigation
