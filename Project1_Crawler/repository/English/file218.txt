      <p>
       The
       <b>
        Wikimedia movement
       </b>
       is the totality of people, activities, and values which revolve around Wikimedia sites and projects:
      </p>
      <ul>
       <li>
        a collection of values shared by individuals (freedom of speech, knowledge for everyone, community sharing, etc.);
       </li>
       <li>
        a collection of projects (
        <a href="/wiki/Wikipedia" title="Wikipedia">
         Wikipedia
        </a>
        ,
        <a href="/wiki/Wiktionary" title="Wiktionary">
         Wiktionary
        </a>
        ,
        <a href="/wiki/Wikimedia_Commons" title="Wikimedia Commons">
         Wikimedia Commons
        </a>
        ,
        <a href="/wiki/Wikinews" title="Wikinews">
         Wikinews
        </a>
        ,
        <a href="/wiki/Wikiquote" title="Wikiquote">
         Wikiquote
        </a>
        ,
        <a href="/wiki/Wikidata" title="Wikidata">
         Wikidata
        </a>
        ,
        <a href="/wiki/Wikivoyage" title="Wikivoyage">
         Wikivoyage
        </a>
        ,
        <a href="/wiki/Wikisource" title="Wikisource">
         Wikisource
        </a>
        ,
        <a href="/wiki/Wikispecies" title="Wikispecies">
         Wikispecies
        </a>
        ,
        <a href="/wiki/Wikiversity" title="Wikiversity">
         Wikiversity
        </a>
        ,
        <a href="/wiki/Wikibooks" title="Wikibooks">
         Wikibooks
        </a>
        ,
        <a class="mw-redirect" href="/wiki/Incubator" title="Incubator">
         Incubator
        </a>
        ...);
       </li>
       <li>
        a collection of activities (conferences, workshops, wikiacademies, etc.);
       </li>
       <li>
        a collection of organizations (
        <a href="/wiki/Special:MyLanguage/Wikimedia_Foundation" title="Special:MyLanguage/Wikimedia Foundation">
         Wikimedia Foundation
        </a>
        ,
        <a class="extiw" href="https://meta.wikimedia.org/wiki/Special:MyLanguage/Wikimedia_chapters" title="m:Special:MyLanguage/Wikimedia chapters">
         Wikimedia chapters
        </a>
        , etc.), as well as some free electrons (individuals without chapters) and similar-minded organizations.
       </li>
      </ul>

      <h2>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Shared_values","replies":[],"headingLevel":2,"placeholderHeading":false}' id="Shared_values">
        <span data-mw-comment-start="" id="h-Shared_values">
        </span>
        Shared values
        <span data-mw-comment-end="h-Shared_values">
        </span>
       </span>
      </h2>
      <ul>
       <li>
        a collection of values shared by individuals (freedom of speech, knowledge for everyone, community sharing, etc.);
       </li>
      </ul>
      <h2>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Shared_projects","replies":[],"headingLevel":2,"placeholderHeading":false}' id="Shared_projects">
        <span data-mw-comment-start="" id="h-Shared_projects">
        </span>
        Shared projects
        <span data-mw-comment-end="h-Shared_projects">
        </span>
       </span>
      </h2>
      <ul>
       <li>
        a collection of activities around
        <a href="/wiki/Special:MyLanguage/Wikimedia_projects" title="Special:MyLanguage/Wikimedia projects">
         Wikimedia projects
        </a>
        :
        <ul>
         <li>
          <a href="/wiki/Special:MyLanguage/Wikipedia" title="Special:MyLanguage/Wikipedia">
           Wikipedia
          </a>
          , in more than 314 language versions,
         </li>
         <li>
          other content related wikis, such as
          <a href="/wiki/Wikisource" title="Wikisource">
           Wikisource
          </a>
          ,
          <a href="/wiki/Wikibooks" title="Wikibooks">
           Wikibooks
          </a>
          or
          <a href="/wiki/Wikidata" title="Wikidata">
           Wikidata
          </a>
          ,
         </li>
         <li>
          'backstage' wikis or organizational wikis, such as this Meta-Wiki,
         </li>
         <li>
          conferences, workshops, wikiacademies etc. (offline activities);
         </li>
        </ul>
       </li>
      </ul>
      <h2>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Wikimedia_organizations","replies":[],"headingLevel":2,"placeholderHeading":false}' id="Wikimedia_organizations">
        <span data-mw-comment-start="" id="h-Wikimedia_organizations">
        </span>
        Wikimedia organizations
        <span data-mw-comment-end="h-Wikimedia_organizations">
        </span>
       </span>
      </h2>
      <ul>
       <li>
        a collection of organizations, as well as some free electrons (individuals acting outside any group) and similar-minded organizations:
        <ul>
         <li>
          <a href="/wiki/Special:MyLanguage/Wikimedia_Foundation" title="Special:MyLanguage/Wikimedia Foundation">
           Wikimedia Foundation
          </a>
          and
          <a href="/wiki/Special:MyLanguage/Wikimedia_Chapters" title="Special:MyLanguage/Wikimedia Chapters">
           Wikimedia Chapters
          </a>
          ,
         </li>
         <li>
          <a href="/wiki/Special:MyLanguage/Wikimedia_Thematic_Organisations" title="Special:MyLanguage/Wikimedia Thematic Organisations">
           Wikimedia Thematic Organisations
          </a>
          ,
         </li>
         <li>
          <a href="/wiki/Special:MyLanguage/Wikimedia_User_Groups" title="Special:MyLanguage/Wikimedia User Groups">
           Wikimedia User Groups
          </a>
          .
         </li>
        </ul>
       </li>
      </ul>
      <h2>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-The_Wikimedia_Foundation","replies":[],"headingLevel":2,"placeholderHeading":false}' id="The_Wikimedia_Foundation">
        <span data-mw-comment-start="" id="h-The_Wikimedia_Foundation">
        </span>
        The Wikimedia Foundation
        <span data-mw-comment-end="h-The_Wikimedia_Foundation">
        </span>
       </span>
      </h2>
      <p>
       The owner of the trademark
       <i>
        Wikimedia
       </i>
       is the
       <a href="/wiki/Special:MyLanguage/Wikimedia_Foundation" title="Special:MyLanguage/Wikimedia Foundation">
        Wikimedia Foundation
       </a>
       (WMF). Linked to the WMF are the Wikimedia Foundation's own organs and committees:
      </p>
      <ul>
       <li>
        board
       </li>
       <li>
        staff
       </li>
       <li>
        advisory board
       </li>
       <li>
        Independent Committees installed by the WMF board
        <ul>
         <li>
          <a href="/wiki/Special:MyLanguage/Affiliations_Committee" title="Special:MyLanguage/Affiliations Committee">
           Affiliations committee
          </a>
         </li>
         <li>
          <a href="/wiki/Special:MyLanguage/Language_committee" title="Special:MyLanguage/Language committee">
           Language committee
          </a>
         </li>
         <li>
          <a href="/wiki/Special:MyLanguage/Funds_Dissemination_Committee" title="Special:MyLanguage/Funds Dissemination Committee">
           Funds Dissemination Committee
          </a>
         </li>
        </ul>
       </li>
      </ul>
      <h2>
       <span class="mw-headline" data-mw-comment='{"type":"heading","level":0,"id":"h-Wikimedia_stakeholders","replies":[],"headingLevel":2,"placeholderHeading":false}' id="Wikimedia_stakeholders">
        <span data-mw-comment-start="" id="h-Wikimedia_stakeholders">
        </span>
        Wikimedia stakeholders
        <span data-mw-comment-end="h-Wikimedia_stakeholders">
        </span>
       </span>
      </h2>
      <p>
       More generally speaking,
       <a class="extiw" href="https://strategy.wikimedia.org/wiki/Wikimedia_stakeholders" title="strategy:Wikimedia stakeholders">
        Wikimedia stakeholders
       </a>
       may also be part of the Wikimedia movement, i.e. readers of Wikimedia content, donors, schools,
       <a href="/wiki/Special:MyLanguage/GLAM" title="Special:MyLanguage/GLAM">
        GLAM
       </a>
       <span style="display:none">
        <a href="/wiki/GLAM" title="GLAM">
        </a>
       </span>
       , similar-minded institutions, and companies co-operating with the Wikimedia Foundation or other Wikimedia organizations.
      </p>
