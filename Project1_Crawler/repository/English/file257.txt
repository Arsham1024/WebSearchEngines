                <td align="left">
                 The instructional design camp is a set of course modules based around a core created by
                 <a href="/wiki/User:Phonebein" title="User:Phonebein">
                  Peter Honebein
                 </a>
                 and students of the
                 <a class="extiw" href="https://en.wikipedia.org/wiki/Indiana_University" title="w:Indiana University">
                  Indiana University
                 </a>
                 . Groups from other universities, as well as other researchers and experts, are welcome to add modules to this project. The goal of this learning experience is to develop the skills needed to design instructional material using media (especially communication technologies) to transfer knowledge as effectively as possible between teachers and learners. The project currently comprises about 130 pages included well-developed modules on
                 <a class="mw-redirect" href="/wiki/ID_learning_objectives" title="ID learning objectives">
                  crafting well-formed learning objectives
                 </a>
                 ,
                 <a class="mw-redirect" href="/wiki/Content_Prioritization" title="Content Prioritization">
                  prioritizing content for learning experiences
                 </a>
                 ,
                 <a href="/wiki/Instructional_Design/Constructivism" title="Instructional Design/Constructivism">
                  designing constructivist learning experiences
                 </a>
                 ,
                 <a class="mw-redirect" href="/wiki/Cognitive_behaviors" title="Cognitive behaviors">
                  designing instruction for cognitive behaviors
                 </a>
                 ,
                 <a class="mw-redirect" href="/wiki/Psychomotor_behaviors" title="Psychomotor behaviors">
                  designing instruction for psychomotor behaviors
                 </a>
                 ,
                 <a class="mw-redirect" href="/wiki/Affective_behaviors" title="Affective behaviors">
                  designing instruction for affective behaviors
                 </a>
                 ,
                 <a class="mw-redirect" href="/wiki/Interpersonal_behaviors" title="Interpersonal behaviors">
                  designing instruction for interpersonal behaviors
                 </a>
                 .
