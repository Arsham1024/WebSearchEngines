         <h1>
          Postorius - Web UI for GNU Mailman
          <a class="headerlink" href="#postorius-web-ui-for-gnu-mailman" title="Permalink to this headline">
           Â¶
          </a>
         </h1>
         <a class="reference external image-reference" href="https://gitlab.com/mailman/postorius/commits/master">
          <img alt="https://gitlab.com/mailman/postorius/badges/master/build.svg" src="https://gitlab.com/mailman/postorius/badges/master/build.svg"/>
         </a>
         <a class="reference external image-reference" href="https://postorius.readthedocs.io">
          <img alt="https://readthedocs.org/projects/postorius/badge" src="https://readthedocs.org/projects/postorius/badge"/>
         </a>
         <a class="reference external image-reference" href="https://mailman.gitlab.io/postorius">
          <img alt="https://gitlab.com/mailman/postorius/badges/master/coverage.svg?job=coverage" src="https://gitlab.com/mailman/postorius/badges/master/coverage.svg?job=coverage"/>
         </a>
         <a class="reference external image-reference" href="https://pypi.python.org/pypi/postorius">
          <img alt="http://img.shields.io/pypi/v/postorius.svg" src="http://img.shields.io/pypi/v/postorius.svg"/>
         </a>
         <a class="reference external image-reference" href="https://pypi.python.org/pypi/postorius">
          <img alt="http://img.shields.io/pypi/dm/postorius.svg" src="http://img.shields.io/pypi/dm/postorius.svg"/>
         </a>
         <p>

         <p>
          The Postorius Django app provides a web user interface to
access GNU Mailman.

          &gt;.
         </p>
         <div class="section" id="requirements">
          <h2>
           Requirements
           <a class="headerlink" href="#requirements" title="Permalink to this headline">
            Â¶
           </a>
          </h2>
          <p>
           Postorius requires Python 3.6+.
          </p>
          <p>
           The minimum Django version is 3.1.
          </p>
          <p>
           Postorius needs a running version of GNU Mailman version 3.3.5.
          </p>
         </div>
         <div class="section" id="installation">
          <h2>
           Installation
           <a class="headerlink" href="#installation" title="Permalink to this headline">
            Â¶
           </a>
          </h2>
          <p>
           To install GNU Mailman Suite follow the instructions in the documentation:
           <a class="reference external" href="https://docs.mailman3.org/">
            https://docs.mailman3.org/
           </a>
          </p>
