           <div class="mp-content-box__header-text">
            Hari ini dalam sejarah
           </div>
          </span>
         </h2>
         <div class="mp-content-box__content" id="mf-hids">
          <p>
           <b>
            <a href="/wiki/30_September" title="30 September">
             30 September
            </a>
           </b>
           :
           <a href="/wiki/Daftar_hari_kemerdekaan_nasional" title="Daftar hari kemerdekaan nasional">
            Hari Kemerdekaan
           </a>
           di
           <b>
            <a href="/wiki/Botswana" title="Botswana">
             Botswana
            </a>
           </b>
           (
           <a href="/wiki/1966" title="1966">
            1966
           </a>
           );
           <b>
            <a class="new" href="/w/index.php?title=Hari_Penistaan&amp;action=edit&amp;redlink=1" title="Hari Penistaan (halaman belum tersedia)">
             Hari Penistaan
            </a>
           </b>
          </p>
          <div style="float:right;margin-left:0.5em;text-align:center; font-style:italic; font-size:90%;">
           <a class="image" href="/wiki/Berkas:Ethernet_RJ45_connector_p1160054.jpg" title="Kabel Eternet">
            <img alt="{{{title}}}" data-file-height="1193" data-file-width="1540" decoding="async" height="116" src="//upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Ethernet_RJ45_connector_p1160054.jpg/150px-Ethernet_RJ45_connector_p1160054.jpg" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Ethernet_RJ45_connector_p1160054.jpg/225px-Ethernet_RJ45_connector_p1160054.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/d/d7/Ethernet_RJ45_connector_p1160054.jpg/300px-Ethernet_RJ45_connector_p1160054.jpg 2x" width="150"/>
           </a>
           <br/>
           Kabel
           <a href="/wiki/Eternet" title="Eternet">
            Eternet
           </a>
          </div>
          <ul>
           <li>
            <b>
             <a href="/wiki/1744" title="1744">
              1744
             </a>
            </b>
            -
            <a href="/wiki/Prancis" title="Prancis">
             Prancis
            </a>
            dan
            <a href="/wiki/Spanyol" title="Spanyol">
             Spanyol
            </a>
            mengalahkan kerajaan
            <b>
             <a href="/wiki/Sardinia" title="Sardinia">
              Sardinia
             </a>
            </b>
            .
           </li>
           <li>
            <b>
             <a href="/wiki/1965" title="1965">
              1965
             </a>
            </b>
            -  Enam pejabat tinggi militer
            <a href="/wiki/Indonesia" title="Indonesia">
             Indonesia
            </a>
            dibunuh dan dibuang di
            <a href="/wiki/Lubang_Buaya" title="Lubang Buaya">
             Lubang Buaya
            </a>
            ,
            <a class="mw-redirect" href="/wiki/Jakarta" title="Jakarta">
             Jakarta
            </a>
            dalam peristiwa
            <b>
             <a href="/wiki/Gerakan_30_September" title="Gerakan 30 September">
              Gerakan 30 September
             </a>
            </b>
            .
           </li>
           <li>
            <b>
             <a href="/wiki/1980" title="1980">
              1980
             </a>
            </b>
            -
            <a href="/wiki/Xerox" title="Xerox">
             Xerox
            </a>
            ,
            <a href="/wiki/Intel" title="Intel">
             Intel
            </a>
            , dan
            <a href="/wiki/Digital_Equipment_Corporation" title="Digital Equipment Corporation">
             Digital Equipment Corporation
            </a>
            menerbitkan spesifikasi pertama
            <b>
             <a href="/wiki/Eternet" title="Eternet">
              Eternet
             </a>
            </b>
            , kemudian menjadi teknologi
            <a href="/wiki/Jaringan_area_lokal" title="Jaringan area lokal">
             jaringan area lokal (LAN)
            </a>
            berkabel yang paling banyak digunakan.
           </li>
           <li>
            <b>
             <a href="/wiki/2005" title="2005">
              2005
             </a>
            </b>
            -
            <a class="mw-redirect" href="/wiki/Surat_kabar" title="Surat kabar">
             Surat kabar
            </a>
            <a href="/wiki/Denmark" title="Denmark">
             Denmark
            </a>
            <a href="/wiki/Jyllands-Posten" title="Jyllands-Posten">
             Jyllands-Posten
            </a>
            memuat
            <b>
             <a href="/wiki/Kontroversi_kartun_Nabi_Muhammad_Jyllands-Posten" title="Kontroversi kartun Nabi Muhammad Jyllands-Posten">
              kartun Nabi Muhammad
             </a>
            </b>
            yang diikuti reaksi keras dari umat
            <a href="/wiki/Muslim" title="Muslim">
             muslim
            </a>
            seluruh dunia.
           </li>
           <li>
            <b>
             <a href="/wiki/2009" title="2009">
              2009
             </a>
            </b>
            -
            <b>
             <a href="/wiki/Gempa_bumi_Sumatra_Barat_2009" title="Gempa bumi Sumatra Barat 2009">
              Gempa bumi
             </a>
            </b>
            berkekuatan 7,6
            <a class="mw-redirect" href="/wiki/Skala_Richter" title="Skala Richter">
             SR
            </a>
            mengguncang
            <a href="/wiki/Sumatra_Barat" title="Sumatra Barat">
             Sumatra Barat
            </a>
            , Indonesia, mengakibatkan lebih dari 1.100 orang tewas dan ribuan lainnya terluka.
           </li>
          </ul>
