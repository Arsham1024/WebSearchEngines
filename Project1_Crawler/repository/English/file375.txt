          <h2 id="mp-tfa-h2" style="margin:0.5em; background:#cef2e0; font-family:inherit; font-size:120%; font-weight:bold; border:1px solid #a3bfb1; color:#000; padding:0.2em 0.4em;">
           <span id=".D4.B1.D5.B4.D5.BD.D5.B8.D6.82.D5.A1.D5.B6_.D5.85.D6.85.D5.A4.D5.B8.D6.82.D5.A1.D5.AE.D5.A8">
           </span>
           <span class="mw-headline" id="Ամսուան_Յօդուածը">
            Ամսուան Յօդուածը
           </span>
          </h2>
          <div id="mp-tfa" style="padding:0.1em 0.6em;">
           <div class="thumb tright">
            <div class="thumbinner" style="width:252px;">
             <a class="image" href="/wiki/%D5%8A%D5%A1%D5%BF%D5%AF%D5%A5%D6%80:Coastline_02.JPG">
              <img alt="" class="thumbimage" data-file-height="1200" data-file-width="1600" decoding="async" height="188" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/ff/Coastline_02.JPG/250px-Coastline_02.JPG" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/ff/Coastline_02.JPG/375px-Coastline_02.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/ff/Coastline_02.JPG/500px-Coastline_02.JPG 2x" width="250"/>
             </a>
             <div class="thumbcaption">
              <div class="magnify">
               <a class="internal" href="/wiki/%D5%8A%D5%A1%D5%BF%D5%AF%D5%A5%D6%80:Coastline_02.JPG" title="Մեծցնել">
               </a>
              </div>
              Սուր, Լիբանան
             </div>
            </div>
           </div>
           <p>
            <b>
             <a href="/wiki/%D5%8D%D5%B8%D6%82%D6%80_(%D6%84%D5%A1%D5%B2%D5%A1%D6%84)" title="Սուր (քաղաք)">
              Սուր
             </a>
            </b>
            (արաբերէն՝ صور, փիւնիկերէն՝ Ṣūr, ասորերէն՝ ܣܘܪ, Tzór, հին յուն․՝ Τύρος Týros, լատ.՝ Tyrus մալակասի՝ Tyrus, Տիր,
            <a href="/wiki/%D5%96%D6%80%D5%A1%D5%B6%D5%BD%D5%A5%D6%80%D5%A7%D5%B6" title="Ֆրանսերէն">
             ֆրանսերէն
            </a>
            ՝ Tyr),
            <a href="/wiki/%D4%BC%D5%AB%D5%A2%D5%A1%D5%B6%D5%A1%D5%B6" title="Լիբանան">
             Լիբանանի
            </a>
            հարաւը գտնուող մեծ քաղաք: Ըստ 2003-ի տուեալներուն, անոր բնակչութեան թիւը 117,000 է: Բայց, Լիբանանի կառավարութիւնը 1932-էն ի վեր շատ յստակ հաշուարկներ չէ հրատարակած, ուստի յստակ վիճակագրութիւն մը կարելի չէ տալ։ Սուր կը գտնուի
            <a href="/wiki/%D5%84%D5%AB%D5%BB%D5%A5%D6%80%D5%AF%D6%80%D5%A1%D5%AF%D5%A1%D5%B6_%D4%BE%D5%B8%D5%BE" title="Միջերկրական Ծով">
             Միջերկրական ծովու
            </a>
            ափին,
            <a href="/wiki/%D5%8A%D5%A7%D5%B5%D6%80%D5%B8%D6%82%D5%A9" title="Պէյրութ">
             Պէյրութէն
            </a>
            80 քիլոմեթր դէպի հարաւ: Տարածութեամբ՝ Լիբանանի չորրորդ մեծագոյն քաղաքն է
            <a href="/wiki/%D5%8A%D5%A7%D5%B5%D6%80%D5%B8%D6%82%D5%A9" title="Պէյրութ">
             Պէյրութէն
            </a>
            ,
            <a href="/wiki/%D4%B9%D6%80%D5%AB%D6%83%D5%B8%D5%AC%D5%AB" title="Թրիփոլի">
             Թրիփոլիէն
            </a>
            եւ
            <a href="/wiki/%D5%8D%D5%A1%D5%B5%D5%BF%D5%A1" title="Սայտա">
             Սայտայէն
            </a>
            ետք: Քաղաքին անունը կը նշանակէ «ժայռ», որ կը վերագրուի քաղաքին շինութեան հիմնաքարերուն:
           </p>
           <p>
            Սուր
            <a href="/wiki/%D5%93%D5%AB%D6%82%D5%B6%D5%AB%D5%AF%D5%A7" title="Փիւնիկէ">
             փիւնիկեան
            </a>
            քաղաք մըն է, ուր ծնած են Եւրոպա եւ Տիտօ դիցաբանութեան հերոսները: Ունի Լիբանանի մեծագոյն նաւահանգիստները: Հոն զարգացած է նաեւ զբօսաշրջութիւնը: Քաղաքը ունի բազմաթիւ պատմական վայրեր, ինչպէս Սուրի հռոմէական բերդը, որ 1979-էն ի վեր
            <a href="/wiki/%D4%B5%D5%88%D5%92%D5%86%D4%B5%D5%8D%D5%94%D5%95" title="ԵՈՒՆԵՍՔՕ">
             ԵՈՒՆԵՍՔՕ
            </a>
            -ի Համաշխարհային Ժառանգութեան Կայքի ցանկին վրայ կը գտնուի:
           </p>
           Սուր կը բաղկանար երկու հեռաւոր բնակավայրերէ՝ Սուրը, որ ծովեզերքը գտնուող կղզի մըն էր եւ ցամաքին վրայ գտնուող Ուշու բնակավայրը:
           <a href="/wiki/%D5%84%D5%A5%D5%AE%D5%B6_%D4%B1%D5%B2%D5%A5%D6%84%D5%BD%D5%A1%D5%B6%D5%BF%D6%80" title="Մեծն Աղեքսանտր">
            Մեծն Աղեքսանտր
           </a>
           Սուրի պաշարման ընթացքին կղզին ցամաքին կը միացնէ ճամբայ մը շինելով ՝ քանդելով հին քաղաքը, անոր քարերը օգտագործելու համար․․․
           <a href="/wiki/%D5%8D%D5%B8%D6%82%D6%80_(%D6%84%D5%A1%D5%B2%D5%A1%D6%84)" title="Սուր (քաղաք)">
            <b>
             Աւելին
            </b>
            <big>
             <b>
              ⇒
             </b>
            </big>
           </a>

          <h2 id="mp-itn-h2" style="margin:0.5em; background:#cedff2; font-family:inherit; font-size:120%; font-weight:bold; border:1px solid #a3b0bf; color:#000; padding:0.2em 0.4em;">

           </span>
           <span class="mw-headline" id="Ի՞նչ_է_Ուիքիփետիան">
            Ի՞նչ է Ուիքիփետիան
           </span>
          </h2>
          <div id="mp-itn" style="padding:0.1em 0.6em;">
           <a href="/wiki/%D5%88%D6%82%D5%AB%D6%84%D5%AB%D6%83%D5%A5%D5%BF%D5%AB%D5%A1" title="Ուիքիփետիա">
            Ուիքիփետիան
           </a>
           ազատ բովանդակութիւն ունեցող, անվճար եւ բազմալեզու առցանց հանրագիտարանային նախագիծ է, որ կը հովանաւորուի
           <a href="/wiki/%D5%88%D6%82%D5%AB%D6%84%D5%AB%D5%B4%D5%A5%D5%BF%D5%AB%D5%A1_%D5%80%D5%AB%D5%B4%D5%B6%D5%A1%D5%A4%D6%80%D5%A1%D5%B4" title="Ուիքիմետիա Հիմնադրամ">
            Ուիքիմետիա Հիմնադրամին
           </a>
           կողմէ։
           <p>
            Ուիքիփետիա անունը կազմուած է «Ուիքի» բառէն (համագործակցաբար կայքէջեր ստեղծելու յատկացուած գործիքի անուն, ինչպէս նաեւ՝ հաուայեան լեզուով «արագ» բառը) եւ
            <a href="/wiki/%D5%85%D5%B8%D6%82%D5%B6%D5%A1%D6%80%D5%A7%D5%B6" title="Յունարէն">
             յունարէն
            </a>
            «էկիքլոփետիա» (հանրագիտարան) բառին «փետիա» (յունարէն լեզուով՝ կրթութիւն) մասէն։
           </p>
           <p>
            Ուիքիփետիա նախագիծը ստեղծուած է
            <a href="/wiki/2001" title="2001">
             2001
            </a>
            -ի Յունուարին,
            <a class="mw-redirect" href="/wiki/%D5%83%D5%AB%D5%B4%D5%B4%D5%AB_%D5%88%D6%82%D5%A7%D5%B5%D5%AC%D5%A6" title="Ճիմմի Ուէյլզ">
             Ճիմմի Ուէյլզի
            </a>
            եւ Լէրի Սէնկըրի կողմէ։ Հանրագիտարանը կը պատկանի Ուիքիմետիա Բարեգործական Հիմնադրամին, որ նուիրատուութիւններէ գոյացած հասոյթը կը յատկացնէ Ուիքիփետիայի եւ այլ քոյր նախագիծերու զարգացման։ Հանրագիտարանին երեսուն միլիոնէ աւելի յօդուածները, որոնց շարքին՝ 7800-է աւելի
            <a href="/wiki/%D4%B1%D6%80%D5%A5%D6%82%D5%B4%D5%BF%D5%A1%D5%B0%D5%A1%D5%B5%D5%A5%D6%80%D5%A7%D5%B6" title="Արեւմտահայերէն">
             արեւմտահայերէնով
            </a>
            յօդուածները ստեղծուած են կամաւոր խմբագիրներու կողմէ։
           </p>
           <a href="/wiki/1_%D4%B1%D5%BA%D6%80%D5%AB%D5%AC" title="1 Ապրիլ">
            1 Ապրիլ
           </a>
           <a class="new" href="/w/index.php?title=2019&amp;action=edit&amp;redlink=1" title="2019 (էջը գոյութիւն չունի)">
            2019
           </a>
           -էն սկսեալ
           <a href="/wiki/%D4%B1%D6%80%D5%A5%D6%82%D5%B4%D5%BF%D5%A1%D5%B0%D5%A1%D5%B5%D5%A5%D6%80%D5%A7%D5%B6_%D5%88%D6%82%D5%AB%D6%84%D5%AB%D6%83%D5%A5%D5%BF%D5%AB%D5%A1" title="Արեւմտահայերէն Ուիքիփետիա">
