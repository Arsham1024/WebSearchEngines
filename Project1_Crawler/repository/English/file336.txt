         <p>
          <a href="/wiki/Krahv" title="Krahv">
           Krahv
          </a>
          <b>
           <a href="/wiki/Alexander_von_Benckendorff" title="Alexander von Benckendorff">
            Konstantin Alexander Karl Wilhelm Christoph von Benckendorff
           </a>
          </b>
          (venepäraselt Александр Христофорович Бенкендорф;
          <a href="/wiki/4._juuli" title="4. juuli">
           4. juuli
          </a>
          (
          <a class="mw-redirect" href="/wiki/Vkj" title="Vkj">
           vkj
          </a>
          23. juuni)
          <a href="/wiki/1781" title="1781">
           1781
          </a>
          <a href="/wiki/Tallinn" title="Tallinn">
           Tallinn
          </a>
          –
          <a href="/wiki/23._september" title="23. september">
           23. september
          </a>
          (vkj. 11. september)
          <a href="/wiki/1844" title="1844">
           1844
          </a>
          <a href="/wiki/Peterburi" title="Peterburi">
           Peterburi
          </a>
          ) oli
          <a href="/wiki/Baltisakslased" title="Baltisakslased">
           baltisaksa
          </a>
          päritolu
          <a href="/wiki/Venemaa_Keisririik" title="Venemaa Keisririik">
           Venemaa
          </a>
          sõjaväelane (
          <a href="/wiki/Jalav%C3%A4ekindral" title="Jalaväekindral">
           jalaväekindral
          </a>
          ) ja riigitegelane. Ta on tuntud Venemaa salapolitsei
          <a class="mw-redirect" href="/wiki/Sandarmi_%C3%9Cksikkorpus" title="Sandarmi Üksikkorpus">
           Sandarmite Erikorpuse
          </a>
          loojana ja
          <a href="/wiki/Riiklik_julgeolek" title="Riiklik julgeolek">
           riikliku julgeolekuasutuse
          </a>
          <a href="/wiki/Tema_Keiserliku_Majesteedi_Isikliku_Kantselei_Kolmas_Osakond" title="Tema Keiserliku Majesteedi Isikliku Kantselei Kolmas Osakond">
           kolmanda osakonna
          </a>
          juhina aastatel
          <a href="/wiki/1826" title="1826">
           1826
          </a>
          –44.
         </p>
         <p>
          <a href="/wiki/1798" title="1798">
           1798
          </a>
          alustas Alexander von Benckendorff väeteenistust
          <a href="/wiki/Allohvitser" title="Allohvitser">
           allohvitserina
          </a>
          <a class="new" href="/w/index.php?title=Semjonovski_ihukaitsev%C3%A4epolk&amp;action=edit&amp;redlink=1" title="Semjonovski ihukaitseväepolk (pole veel kirjutatud)">
           Semjonovski ihukaitseväepolgus
          </a>
          . Sama aasta detsembris oli ta juba
          <a href="/wiki/Prapor%C5%A1t%C5%A1ik" title="Praporštšik">
           praporštšik
          </a>
          ja sai
          <a href="/wiki/Paul_I" title="Paul I">
           Paul I
          </a>
          <a href="/wiki/Tiibadjutant" title="Tiibadjutant">
           tiibadjutandiks
          </a>
          .
          <a href="/wiki/1803" title="1803">
           1803
          </a>
          osales ta sõjas
          <a href="/wiki/Kaukaasia" title="Kaukaasia">
           Kaukaasias
          </a>
          , kus paistis silma vaprusega. Ta osales sõjakäigul ka
          <a href="/wiki/1806" title="1806">
           1806
          </a>
          –07, sealhulgas
          <a class="new" href="/w/index.php?title=Eylau_lahing&amp;action=edit&amp;redlink=1" title="Eylau lahing (pole veel kirjutatud)">
           Eylau lahingus
          </a>
          .
          <a href="/wiki/1807" title="1807">
           1807
          </a>
          ülendati ta
          <a href="/wiki/Polkovnik" title="Polkovnik">
           polkovnikuks
          </a>
          . Ta osales
          <a href="/wiki/Vene-T%C3%BCrgi_s%C3%B5da_(1806%E2%80%931812)" title="Vene-Türgi sõda (1806–1812)">
           Vene-Türgi sõjas
          </a>
          .
          <a href="/wiki/1811" title="1811">
           1811
          </a>
          lõi ta
          <a class="new" href="/w/index.php?title=Ruse_lahing&amp;action=edit&amp;redlink=1" title="Ruse lahing (pole veel kirjutatud)">
           Ruse lahingus
          </a>
          tagasi Venemaa vägede tagalat ohustanud
          <a href="/wiki/Osmanite_riik" title="Osmanite riik">
           Türgi
          </a>
          väed, mille eest ta ülendati
          <a href="/wiki/Kindralmajor" title="Kindralmajor">
           kindralmajoriks
          </a>
          ja teda autasustati
          <a href="/wiki/P%C3%BCha_Georgi_orden" title="Püha Georgi orden">
           Püha Georgi 4. järgu ordeniga
          </a>
          .
          <b>

         <p>
          <b>
           <a href="/wiki/Vikipeedia" title="Vikipeedia">
            Vikipeedia
           </a>
          </b>
          on
          <a class="extiw" href="https://meta.wikimedia.org/wiki/List_of_Wikipedias#All_Wikipedias_ordered_by_number_of_articles" title="meta:List of Wikipedias">
           paljukeelne
          </a>
          <a href="/wiki/Vikipeedia:Sissejuhatus" title="Vikipeedia:Sissejuhatus">
           võrguentsüklopeedia
          </a>
          , mida kõik saavad vabalt kasutada ning täiendada ja parandada.
         </p>
         <p>
          Esimesena alustas 2001. aasta jaanuaris
          <a href="/wiki/Ingliskeelne_Vikipeedia" title="Ingliskeelne Vikipeedia">
           ingliskeelne Vikipeedia
          </a>
          .
          <a href="/wiki/Eestikeelne_Vikipeedia" title="Eestikeelne Vikipeedia">
           Eestikeelne Vikipeedia
          </a>
          alustas 2002. aasta augustis.
         </p>
         <p>
          <b>
           <span style="color:#002BB8">
            Sinised
           </span>
          </b>
          lingid viivad olemasolevate artikliteni;
          <b>
           <span style="color:#BA0000">
            punaste
           </span>
          </b>
          linkide puhul on artikkel ette nähtud, kuid see tuleb alles kirjutada. Punasele lingile klõpsates jõuate lehele, millel saate alustada uut artiklit.
         </p>
         <p>
          <a href="/wiki/Vikipeedia:Tehke_julgelt_parandusi" title="Vikipeedia:Tehke julgelt parandusi">
           Tee julgelt parandusi
          </a>
          ! Muudatused on kohe nähtavad, sisselogimine ei ole kohustuslik. Vaadake
          <a href="/wiki/Juhend:Lehek%C3%BClje_muutmine" title="Juhend:Lehekülje muutmine">
           redigeerimisjuhendit
          </a>
          ,
          <a href="/wiki/Vikipeedia:KKK" title="Vikipeedia:KKK">
           korduma kippuvaid küsimusi
          </a>
          ja
          <a href="/wiki/Juhend:Sisukord" title="Juhend:Sisukord">
           teisi abilehti
          </a>
          ning hakake kirjutama! Harjutamiseks võib kasutada ka
          <a href="/wiki/Vikipeedia:Liivakast" title="Vikipeedia:Liivakast">
           liivakasti
          </a>
          . Kirjutamisel soovitame eeskujuks võtta siinse kogukonna poolt
          <a href="/wiki/Vikipeedia:Head_artiklid" title="Vikipeedia:Head artiklid">
           headeks valitud artikleid
          </a>
          .
         </p>
         <p>
          Vahepeal tehtut näeb
          <a href="/wiki/Eri:Viimased_muudatused" title="Eri:Viimased muudatused">
           viimaste muudatuste lehelt
          </a>
          .
         </p>
         <p>
          Kui artiklis jäi midagi arusaamatuks, siis postitage oma küsimused vastava artikli arutelulehele, kuhu pääsete vastava artikli päisest. Arutelulehele postitamiseks kasutage päiselinki "
          <span style="color:#002bb8;font-size:0.9em;">
           Muuda
          </span>
          " või "
          <span style="color:#002bb8;font-size:0.9em;">

          </span>
          ".
