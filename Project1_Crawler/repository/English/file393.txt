          <div style="font-size:9pt; padding:5px; background-color: transparent">
           <ul>
            <li>
             Wikipedia ashıq ensiklopediya bolıp, ha'r bir qa'lewshi bul ensiklopediyada qatnasıp o'zinin' u'lesin qosıwı mu'mkin.
            </li>
            <li>
             Eger Wikipedia proektlerinde birinshi ret qatnasıp atırg'an bolsan'ız,
             <a href="/wiki/Wikipedia:Maqala_baslawsh%C4%B1lar_ush%C4%B1n_q%C4%B1sqa_qollanba" title="Wikipedia:Maqala baslawshılar ushın qısqa qollanba">
              Baslawshılar ushın qısqa qollanba
             </a>
             betinen baslawın'ızdı ma'slahat beremiz.
            </li>
            <li>
             Eger Wikipediada maqala jaratıwdın' qag'ıydaları menen tanıs bolsan'ız, biraq qanday maqala jazıwdı bilmey tursan'ız,
             <a href="/wiki/Wikipedia:Ha%27mme_Wikipediyalarda_bol%C4%B1w%C4%B1_tiyis_maqalalar_dizimi" title="Wikipedia:Ha'mme Wikipediyalarda bolıwı tiyis maqalalar dizimi">
              Ha'mme Wikipediyalarda bolıwı tiyis maqalalar dizimi
             </a>
             menen tanısıp shıg'ın', sol jerden ba'lkim o'zin'izge qızıq bolg'an maqalalardı tabarsız.
            </li>
            <li>
             Maqala jaratqanda ishindegi mag'lıwmattı tek qaraqalpaq tilinde jazın', basqa tillerde jazıw qadag'an etiledi ha'm olar administratorlar ta'repinen sorawsız o'shiriledi.
            </li>
            <li>
             Wikipediag'a u'les qosıwdı baslawın'ızdan aldın o'zin'izge akkaunt jaratıwın'izdı ma'slahat etemiz.
            </li>
            <li>
             Maqala jaratıw ushın to'mendegi tekst maydanına maqalan'izdın' atamasın jazıp, "Bet jaratıw"dı basın'. Biraq maqala jaratıwdan aldın bul maqala a'lle qashan jaratılg'anba ekenligin tekserip ko'rin. Onın' ushın usı bettin' shep jag'ında jaylasqan "izlew" bo'limindegi tekst maydanına maqala atamasın jazıp, "İzle"ni basın'.
            </li>
           </ul>

            <b>
             Ser Isaac Newton
            </b>
            (4-Yanvar' 1643 – 31-Mart 1727) -
            <a href="/wiki/Angliya" title="Angliya">
             Angliyalı
            </a>
            <a href="/wiki/Fizik" title="Fizik">
             fizik
            </a>
            ,
            <a href="/wiki/Matematik" title="Matematik">
             matematik
            </a>
            ,
            <a href="/wiki/Astronom" title="Astronom">
             astronom
            </a>
            ,
            <a class="new" href="/w/index.php?title=Ta%27biyiy_filosof&amp;action=edit&amp;redlink=1" title="Ta'biyiy filosof (bet ele jaratılmag'an)">
             ta'biyiy filosof
            </a>
            ,
            <a class="new" href="/w/index.php?title=Alximik&amp;action=edit&amp;redlink=1" title="Alximik (bet ele jaratılmag'an)">
             alximik
            </a>
            ,
            <a class="new" href="/w/index.php?title=Teolog&amp;action=edit&amp;redlink=1" title="Teolog (bet ele jaratılmag'an)">
             teolog
            </a>
            ha'm adamzat tariyxinda en' a'hmiyetli insanlardin' biri. Onın' 1687-jılı basılıp shiqqan
            <a class="new" href="/w/index.php?title=Philosophi%C3%A6_Naturalis_Principia_Mathematica&amp;action=edit&amp;redlink=1" title="Philosophiæ Naturalis Principia Mathematica (bet ele jaratılmag'an)">
             Philosophiæ Naturalis Principia Mathematica
            </a>
            (Tabiyiy
            <a href="/wiki/Filosofiya" title="Filosofiya">
             filosofiyanın'
            </a>
            matematikalıq printsipleri) kitabı
            <a class="new" href="/w/index.php?title=%C4%B0lim_tariyx%C4%B1&amp;action=edit&amp;redlink=1" title="İlim tariyxı (bet ele jaratılmag'an)">
             ilim tariyxında
            </a>
            en' a'hmiyetli kitap esaplanadı. Bul kitabında Newton
            <a class="new" href="/w/index.php?title=Dunya-ju%27zlik_tart%C4%B1l%C4%B1s%C4%B1w_(gravitatsiya)&amp;action=edit&amp;redlink=1" title="Dunya-ju'zlik tartılısıw (gravitatsiya) (bet ele jaratılmag'an)">
             dunya-ju'zlik tartılısıw (gravitatsiya)
            </a>
            ha'm
            <a class="new" href="/w/index.php?title=Qozg%27al%C4%B1s&amp;action=edit&amp;redlink=1" title="Qozg'alıs (bet ele jaratılmag'an)">
             qozg'alıstın'
            </a>
            u'sh
            <a class="new" href="/w/index.php?title=Nizam&amp;action=edit&amp;redlink=1" title="Nizam (bet ele jaratılmag'an)">
             nizamın
            </a>
            sipatlap berdi. Bul jumisı menen ol zamanag'oy injener isinin' tiykarı ha'm kelesi u'sh a'sir dawamında fizikalıq du'nyag'a ilimiy ko'z qarastı belgilep bergen
            <a class="new" href="/w/index.php?title=Klassik_mexanika&amp;action=edit&amp;redlink=1" title="Klassik mexanika (bet ele jaratılmag'an)">
             klassik mexanikanın'
            </a>
            tiykarın saldı. Newton jerdegi ob'ektlerdin' ha'm
            <a class="new" href="/w/index.php?title=Aspan_deneleri&amp;action=edit&amp;redlink=1" title="Aspan deneleri (bet ele jaratılmag'an)">
             aspan denelerinin'
            </a>
            ha'reketleri birdey ta'biyiy nızamlarg'a boysınatug'ınlıg'ın o'zinin'
            <a class="new" href="/w/index.php?title=Dunya-ju%27zlik_tartilisiw_(gravitatsiya)&amp;action=edit&amp;redlink=1" title="Dunya-ju'zlik tartilisiw (gravitatsiya) (bet ele jaratılmag'an)">
             gravitatsiya
            </a>
            teoriyası menen Keplerdin' planetar qozg'alıs nizamları sa'ykesligin da'lillewi menen ko'rsetip berdi. Bul arqali ol
            <a class="new" href="/w/index.php?title=Geliotsentrizm&amp;action=edit&amp;redlink=1" title="Geliotsentrizm (bet ele jaratılmag'an)">
             geliotsentrizm
            </a>
            haqqındag'i aqırg'ı isenbewshiliklerdi joq etti ha'm
            <a class="new" href="/w/index.php?title=%C4%B0limiy_revolyutsuya&amp;action=edit&amp;redlink=1" title="İlimiy revolyutsuya (bet ele jaratılmag'an)">
             ilimiy revolyutsuyanı
            </a>
            tezletti.
            <i>
             <a href="/wiki/Isaac_Newton" title="Isaac Newton">
              dawamı...
             </a>
